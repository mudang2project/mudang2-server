• 가천대 무당이 셔틀

1. '가천대 무당이 셔틀' App이란?

학생들이 캠퍼스 내에서 이동할 때 탑승하는 교내 셔틀버스(이하 '무당이')를 더 편하게 이용하는 데에 도움이 되고자 고안하게 된 어플리케이션 서비스입니다.

2. 누가 만들었나요?

컴퓨터공학과 3, 4학년 학생이 만들었어요. 무당이가 어느 정거장을 지나고 있는지, 정거장의 대기인원은 얼마나 되는지를 알 수 있다면 좋겠다는 생각으로 개발하게 된 졸업작품입니다. 이후 대학이 우수작으로 선정하고 지원하게 되어, 드디어 많은 사람들이 이용할 수 있게 되었어요.

3. 어떤 기능이 있나요?

① 무당이의 위치 확인

지도 상에 운행 중인 무당이의 위치를 표시해줍니다.

② 정류장 대기인원 확인

산학협력관2 건물(구 IT융합대학 건물) 앞 정류장 대기인원 수를 표시해줍니다.
대기인원이 5명 이하인 경우 '원활', 6명 이상 15명 이하인 경우 '보통', 16명 이상일 시 '혼잡'으로 표시합니다.

③ 날씨 확인

날씨에 따라 운행 여부가 결정되는 무당이의 특성을 고려하여, 복정동의 현재 날씨와 기온을 확인할 수 있습니다.

4. 어떻게 이용하나요?

'가천대 무당이 셔틀' App을 스토어에서 설치하면, 별도의 회원가입 없이 어플에 접속하는 것만으로도 서비스를 이용하실 수 있습니다.
단, 서비스 특성상 인터넷 환경에 연결되어 있어야 이용이 가능합니다.
